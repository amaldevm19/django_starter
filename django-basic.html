Django Web Project

  • Download and install VS code editor from https://code.visualstudio.com/download
  • Download and install latest Python version from https://www.python.org/downloads/
  • Check your path in environment variable
  • Open cmd / Power shell and run
    •         python --version
    
        pip --version
  • Install pipenv using Instructions 
    •         pipenv --version
  • Create new workfolder
  • Activate virtual environment in your work folder
  • Open your workfolder in power shell
    •         pipenv shell
  • Install Django
    •         pip install django
  • Start new Django project
    •         django-admin startproject django_starter
  • Change directory to 
    •         cd django_starter
  • Open project in your code editor
    •         code .
  • Run below command in your terminal window
    •         python manage.py runserver
  • Check http://127.0.0.1:8000/
  • To stop server
    •         ctrl + c
